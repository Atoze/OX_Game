# OX_Game　[![CircleCI](https://circleci.com/gh/Atoze/OX_Game.svg?style=svg)](https://circleci.com/gh/Atoze/OX_Game)

## Description
簡易的な○×ゲームです.

## Requirement
Java 1.8.0

## HowToPlay
1. ./gradlew runで実行できます.
2. CPUと交互に0から8までの数字を入力し、相手より先に一列を作ることができれば勝利です.
